# To-Do List Manager (Python)

This is a simple command-line To-Do List application made in Python.
It allows users to add tasks, view them, mark tasks as completed, and delete them.
All tasks are saved in a text file so they stay even after the program is closed.

---

## Features

* Add new tasks
* View existing tasks
* Mark tasks as completed
* Delete tasks
* Saves tasks in `tasks.txt` using file handling

---

## Files Included

* **todo.py** – main program
* **tasks.txt** – created automatically to store tasks

---

## How to Run

1. Open the terminal in the project folder
2. Run the command:

```
python todo.py
```

Use `python3 todo.py` if Python 3 is required on your system.

---

## Menu Options

```
1. View Tasks
2. Add Task
3. Complete Task
4. Delete Task
5. Exit
```
